## 编写第一个ts：hello world

+ 新建文件夹basic并初始化npm
```linux
    mkdir basic
    npm init -y
```
+ 全局安装ts,安装过的不需要安装
```linux
    npm install -g typescript
```

+ 创建ts配置项,即可创建tsconfig.json文件
```linux
    tsc --init
```
